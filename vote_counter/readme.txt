We are going to hold a vote on the election of the Board of Directors! Ben, who is in charge of the UN computer, 
has written a program that counts how many votes each country has received. You are going to help Ben count the 
votes by writing a program.

The first line of input contains the number n, which represents the total number of votes. Each of the next n 
lines contains the name of a country. The names of the countries are made up of lowercase English letters.

Print m lines of output containing the number of votes each country has received. Write the names of the countries 
in alphabetical order in the output. For more information, see sample Input and sample Output.

for example:

input:
5
iran
spain
germany
iran
germany

output:

germany 2
iran 2
spain 1
